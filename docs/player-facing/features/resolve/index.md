## Description
&lt;p class="Card-Feature"&gt;You gain a pool of dice called &lt;strong&gt;Bullet dice&lt;/strong&gt;. Whenever you fail an attack roll with Fear, add a d6 to your Bullet die pool. You can store a number of Bullet die equal to your proficiency. When you make an attack roll or damage roll, you can spend any number of these Bullet Dice, rolling them and adding their result to the total. At the start of each session, clear any unspent Bullet Dice and gain 1 Hope per die cleared.&lt;/p&gt;
