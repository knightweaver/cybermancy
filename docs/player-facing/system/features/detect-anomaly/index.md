# Detect anomaly
<img src="../../../assets/icons/features/detect-anomaly.webp" alt="Detect anomaly" class="item-image" style="width:300px; height:auto;">
## Description
*You are highly attuned to the presence of effects that attempt to alter the local Resonance.  Spend an action concentrating to detect if there are any anomalies within Close.  Spend at least 1 minute concentrating to detect information about any anomalies within Very Close – you may ask the GM up to 3 yes/no questions about an resonance field disturbance you detect and examine in this way.*
