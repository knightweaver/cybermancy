# Stillpoint
<img src="../../../assets/icons/features/stillpoint.webp" alt="Stillpoint" class="item-image" style="width:300px; height:auto;">
## Description
*Spotlight an action to enter a quantum superposition state. While in Stillpoint, you cannot attack or move more than Very Close, but all allies within Very Close gain +2 Evasion and advantage on Reactions to spells, magical effects, and resonance-based effects from the Environment.  The state ends if you take damage or choose to release it.*
