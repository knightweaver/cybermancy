# Coming through
<img src="../../../assets/icons/features/coming-through.webp" alt="Coming through" class="item-image" style="width:300px; height:auto;">
## Description
*<p class="Card-Feature">Once per rest, <strong>Mark 1 Stress</strong> per ICE on a Device you are attempting to Control. Make a <strong>Spellcasting (Hacking)</strong> roll against each ICE at +2. Any ICE you succeed against are removed from the Device. Raise the System Alert by 1 for each ICE you fail against. You cannot stop rolling until all the ICE have been rolled against.</p>*
