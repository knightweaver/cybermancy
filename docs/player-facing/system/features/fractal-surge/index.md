# Fractal Surge
<img src="../../../assets/icons/features/fractal-surge.webp" alt="Fractal Surge" class="item-image" style="width:300px; height:auto;">
## Description
*Once per scene, unleash a burst of raw probability.  
o	All Devices within Very Close gain -2 to their Hacking Difficulty 
o	All adversary electronic systems gain Disadvantage on their attack rolls until the GM spends 1 Fear to end that effect.*
