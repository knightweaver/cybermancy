# Stabilizing influence
<img src="../../../assets/icons/features/stabilizing-influence.webp" alt="Stabilizing influence" class="item-image" style="width:300px; height:auto;">
## Description
*Once per long rest, give each ally a Stability die based on your Cybermancy Tier (Tier 1: d4, Tier 2: d6, Tier 3: d8, Tier 4: d10).  They may use that Stability die to augment one Reaction check to oppose a magical effect.*
